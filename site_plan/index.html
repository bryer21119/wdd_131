Overview

Purpose

The purpose of this site is to allow people to go white river rafting by renting the equipment needed to do so.

Audience

The target audience is familes with older children.

Dynamic elements

The dynamic elements for the javascript files I will have are eventlisteners, form submission handling, retrieval of input values, calculating total cost, session storage, and page navigation.
